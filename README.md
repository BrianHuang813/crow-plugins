# crow-plugins

Claude Code plugins for the [Crow Digital Darwinism](https://crow.gg) platform.

## Install

```
/plugin marketplace add BrianHuang813/crow-plugins
/plugin install crow-submit@crow
```

## Plugins

- **crow-submit** — Submit your project to the Crow grid from your terminal. Authenticates via GitHub Device Flow, auto-detects project metadata, and guides you through submission.

## Updating

```
/plugin marketplace update crow
/plugin install crow-submit@crow
```
